# Privacy

Lenia Life Lab does not collect analytics in v1.

## Data Stored Locally

The app can store saved species and the audio preference in this browser's `localStorage`.

Stored species fields:

- Species ID
- Name
- Seed
- Lenia parameters
- Colorway
- Creation time
- Parent IDs

This data stays on the user's device unless they copy or share a URL hash.

## Data In URLs

Shared species are encoded in the URL hash after `#species=`. The hash contains the compact species genome, not an account, email, IP address, or secret.

## Network Requests

The app loads static assets from https://baditaflorin.github.io/lenia-life-lab/ and links out to:

- https://github.com/baditaflorin/lenia-life-lab
- https://www.paypal.com/paypalme/florinbadita

No metrics beacon, tracking pixel, or server-side event collector is included.
